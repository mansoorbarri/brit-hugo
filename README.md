# brit-hugo
A hugo theme inspired by brittanychiang.com

## Setup 

make a test site 
```
hugo new site test-brithugo
```

cd & clone the theme 
```
cd && test-brithugo && git clone https://github.com/mansoorbarri/brit-hugo themes/brit-hugo
```

copy `config.toml`
```
cp themes/brit-hugo/examplesite/config.toml . 
``` 

run at localhost 
```
hugo server 
```
